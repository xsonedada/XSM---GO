# 🔐 XSM — eXtreme Secure Messenger \\ XSONE Secure Messenger

<p align="center">
  <b>Современный защищённый мессенджер</b> с красивым дизайном, мгновенной доставкой сообщений и заботой о приватности.
</p>

---

## ✨ Возможности

### 🔐 **Безопасность**
- End-to-end шифрование (AES-256 + RSA-4096)
- JWT-аутентификация и refresh-токены
- Хеширование паролей bcrypt

### 💬 **Чаты и сообщения**
- Личные чаты с поиском пользователей
- Мгновенная доставка через WebSocket
- Ответы на сообщения (цитирование)
- Удаление сообщений и чатов
- Разделители по датам

### 📊 **Статусы и уведомления**
- Статусы сообщений: отправлено ✓, доставлено ✓✓, прочитано ✓✓ (синие)
- Счётчик непрочитанных в заголовке страницы и на чатах
- Браузерные уведомления (когда вкладка неактивна)
- Индикатор онлайн у собеседника

### 🧑‍💻 **Профили пользователей**
- Аватарки (загрузка с компьютера)
- Статусное сообщение
- Биография «О себе»
- Настройки видимости (заглушка)

### 🎨 **Дизайн**
- Тёмная тема
- Адаптивная вёрстка под любые устройства
- Анимированные переходы, контекстные меню

## 🛠️ Технологии

| Компонент | Стек |
|-----------|------|
| **Бэкенд** | Go 1.21, Gin, Gorilla WebSocket |
| **База данных** | PostgreSQL 15+ |
| **Аутентификация** | JWT (access + refresh), bcrypt |
| **Шифрование** | RSA-4096 + AES-256 (серверное хранилище ключей, клиентское E2E запланировано) |
| **Фронтенд** | Vanilla JS, HTML5, CSS3, Font Awesome 6 |
| **Real-time** | WebSocket (gorilla/websocket) |

## 📦 Установка и запуск

### Предварительные требования
- Go 1.21+
- PostgreSQL 15+
- Git

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/xsonedada/XSM---GO.git
cd xsm-messenger
```
2. Настройте базу данных
Создайте пользователя и базу в PostgreSQL:

```
CREATE USER xsm_user WITH PASSWORD 'ваш_пароль';
CREATE DATABASE xsm_db OWNER xsm_user;
```
Или используйте свои параметры – они будут указаны в .env.

3. Настройте переменные окружения
Скопируйте пример конфигурации и отредактируйте под себя:

```bash
cp .env.example .env
Откройте .env и укажите реальные данные:
```

```bash
DB_HOST=localhost
DB_PORT=5432
DB_USER=xsm_user
DB_PASSWORD=ваш_пароль
DB_NAME=xsm_db
JWT_SECRET=случайная_длинная_строка
ENCRYPTION_KEY=32_байта_ключа_для_AES
Остальные параметры можно оставить по умолчанию.
```

4. Установите зависимости Go
```bash
go mod tidy
```
5. Запустите сервер
```bash
go run ./cmd/server/main.go
```
Сервер автоматически выполнит миграции (создаст таблицы) при первом запуске.

6. Откройте мессенджер
Перейдите в браузере по адресу:
👉 http://localhost:8080

Зарегистрируйтесь, войдите и начинайте общаться!

📁 Структура проекта
```
xsm/
├── cmd/server/          # Точка входа, main.go
├── internal/
│   ├── auth/            # Аутентификация, JWT, middleware
│   ├── config/          # Загрузка конфигурации из .env
│   ├── database/        # Подключение к PostgreSQL, миграции
│   ├── encryption/      # AES, RSA, E2E-сервис
│   ├── handlers/        # Обработчики API (чаты, сообщения, пользователи)
│   ├── models/          # Структуры данных
│   └── websocket/       # WebSocket-хаб и клиент
├── web/
│   ├── css/style.css    # Все стили (премиум тёмная тема)
│   ├── js/              # Логика фронтенда (auth.js, messenger.js)
│   ├── auth.html        # Страница авторизации
│   └── messenger.html   # Главная страница мессенджера
├── .env.example         # Пример конфигурации
├── go.mod / go.sum
├── Dockerfile
├── docker-compose.yml   # (опционально) для быстрого деплоя
└── README.md
```
🚀 Быстрый старт с Docker
Если лень ставить Go и PostgreSQL отдельно, можно запустить всё одной командой:

```bash
docker compose up -d
```
Сервер будет доступен на http://localhost:8080.
